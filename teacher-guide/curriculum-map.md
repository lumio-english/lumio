# Lumio English — Full Curriculum Map (11 levels × 20 lessons)

Age guide: Pre-A/Pre-B → 4–6 · L1–L3 → 6–8 · L4–L6 → 8–10 · L7–L9 → 10–12+
CEFR guide: Pre-A/Pre-B → pre-A1 · L1–L2 → A1 · L3–L4 → A1+/A2 · L5–L6 → A2 · L7–L8 → A2+/B1 · L9 → B1

Each lesson = new vocab (5–9 items) + listen-choose + match + quiz + spell. From Level 1 up, add sentence-building; from Level 3 up, add short reading passages; from Level 5 up, add writing prompts and grammar focus boxes.

---
**To generate a new level:** copy `gen_prea.py`'s pattern — each lesson needs `title`, `titleAr`, `goal`, `vocab[]` (en/ar/emoji/example), optional custom `quiz[]`. Drop the JSON files into `lessons/{level-id}/` and add the level id to `AVAILABLE_LEVELS` in `js/app.js`. The lesson engine handles the rest automatically.
